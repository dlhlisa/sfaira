sfaira.data.DatasetInteractive.write_backed¶

DatasetInteractive.write_backed(adata_backed: anndata._core.anndata.AnnData, genome: str, idx: numpy.ndarray, load_raw: bool = False, allow_caching: bool = True)¶

Loads data set into slice of backed anndata object.

Note: scatter updates to backed sparse arrays are not yet supported by anndata. Accordingly, we need to work around below using .append() of the backed matrix.

Parameters

  • adata_backed –

  • genome – Genome name to use as refernce.

  • idx – Indices in adata_backed to write observations to. This can be used to immediately create a shuffled object.

  • load_raw – See .load().

  • allow_caching – See .load().

Returns

New row index for next element to be written into backed anndata.
